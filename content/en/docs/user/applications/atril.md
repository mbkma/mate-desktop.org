Atril is the official document viewer for the MATE desktop. It supports a wide range of document formats and is suitable for both everyday reading and presentations.

![Atril window](/images/applications/atril-window.png "Atril document viewer")

## Supported Formats

| Format | Notes |
|--------|-------|
| PDF | Portable Document Format |
| PostScript / EPS | PS and Encapsulated PostScript |
| DjVu | Scanned document format |
| DVI | TeX output format |
| XPS | XML Paper Specification |
| TIFF | Multi-page TIFF images |
| CBR / CBZ | Comic book archives |

## Features

- **Continuous and dual-page view** — read documents in single, dual, or continuous scroll mode
- **Presentation mode** — fullscreen slideshow for PDF presentations
- **Thumbnails sidebar** — quickly jump to any page
- **Bookmarks** — save positions within a document
- **Text search** — find text across the entire document
- **Copy to clipboard** — select and copy text from supported documents
- **Hyperlink navigation** — clickable links within documents
- **Printing** — print to any configured printer
