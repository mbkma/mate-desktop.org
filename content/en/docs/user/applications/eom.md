Eye of MATE (eom) is the official image viewer for the MATE desktop. It is designed to be fast and lightweight while handling large images efficiently. It uses the gdk-pixbuf library and can display a wide range of image formats. Basic editing operations such as rotation and flipping are also supported.

![Eye of MATE window](/images/applications/eom-window.png "Eye of MATE image viewer")

## Supported Formats

- JPEG, PNG, GIF, BMP, TIFF, ICO
- SVG (with librsvg)
- WebP, AVIF (depending on gdk-pixbuf loaders installed)
- Many additional formats via plugins

## Features

- **Slideshow** — automatically advance through images in a folder
- **Full-screen mode** — view images without distractions
- **Zoom and pan** — smooth zoom from fit-to-window to 1:1 and beyond
- **Rotation and flip** — rotate 90°/180°/270° and flip horizontally or vertically
- **EXIF metadata** — view camera settings, GPS coordinates, and other metadata embedded in JPEG files
- **Image collection panel** — thumbnail strip for navigating images in a folder
- **Plugin system** — extend with features such as map view, fullscreen background, and more
