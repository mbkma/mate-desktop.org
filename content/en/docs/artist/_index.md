## Submitting contributions

All artwork repositories follow the same process as code contributions:

1. Fork the repository on GitHub.
2. Add your work and update the relevant `Makefile.am` / `meson.build`.
3. Open a pull request with a screenshot and licence information.

| What | Repository |
|------|-----------|
| Wallpapers | [mate-backgrounds](https://github.com/mate-desktop/mate-backgrounds) |
| GTK & window themes | [mate-themes](https://github.com/mate-desktop/mate-themes) |
| Icon themes | [mate-icon-theme](https://github.com/mate-desktop/mate-icon-theme) |
