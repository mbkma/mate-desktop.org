## Chat

### Matrix

Join us on Matrix at [#mate-desktop-environment:matrix.org](https://matrix.to/#/%23mate-desktop-environment:matrix.org).

**Recommended clients**

  {{< tab name="Linux" >}}

  Install via Flathub: `flatpak install flathub io.element.Element`
- [**Fractal**](https://flathub.org/apps/org.gnome.Fractal) — lightweight GNOME client, fits naturally on the MATE desktop.
  Install via Flathub: `flatpak install flathub org.gnome.Fractal`

## Code of Conduct

The MATE community is committed to providing a welcoming and inclusive environment for everyone. We expect all community members to:

- Be respectful and considerate
- Be patient with newcomers
- Focus on constructive feedback
- Support and help fellow community members

## Getting Help

If you need help with MATE:

1. Check the [Documentation](/docs)
2. Search existing issues on GitHub
3. Ask on Matrix
4. Post on distribution forums
