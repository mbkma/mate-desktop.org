## Verifying Downloads

Each tarball has a corresponding SHA256 checksum file on GitHub. To verify:

```bash
wget https://github.com/mate-desktop/mate-desktop/releases/download/v1.28.2/mate-desktop-1.28.2.tar.xz
wget https://github.com/mate-desktop/mate-desktop/releases/download/v1.28.2/mate-desktop-1.28.2.tar.xz.sha256sum
sha256sum -c mate-desktop-1.28.2.tar.xz.sha256sum
```
