You can contribute to MATE without writing any code by creating wallpapers and GTK themes.

## Wallpapers

Good wallpapers are calm and unobtrusive — many people look at their desktop for hours at a time. MATE-branded imagery is always welcome.

Useful open-source tools:

- [GIMP](https://www.gimp.org/) — raster image editor (photos, compositing)
- [Inkscape](https://inkscape.org/) — vector image editor (crisp at any resolution)

Submit finished wallpapers to the [mate-backgrounds](https://github.com/mate-desktop/mate-backgrounds) repository.

## GTK Themes

MATE uses the [GTK 3](https://docs.gtk.org/gtk3/) toolkit. When designing a theme, the GNOME Human Interface Guidelines 2.2.1 remain a useful reference for MATE's visual conventions:

- [GNOME HIG 2.2.1](https://mbkma.github.io/gnome-2-hig-book/)

Submit new themes or improvements to the [mate-themes](https://github.com/mate-desktop/mate-themes) repository.
