Translating MATE into your language is one of the easiest ways to contribute — no programming knowledge required.

MATE uses [Transifex](https://www.transifex.com) to manage translations. All application strings, desktop file metadata, and help content are available there.

## How to get started

1. Create a free account at [transifex.com/signup](https://www.transifex.com/signup/).
2. Open the [MATE project page](https://www.transifex.com/projects/p/MATE/).
3. Find your language team and click **Join team**. If your language does not exist yet, you can request it.
4. Pick a resource to translate and start suggesting strings.

Translations are pulled into the repositories periodically before each release.
