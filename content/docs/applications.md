MATE Desktop includes a suite of applications to handle common tasks. These applications are designed to be lightweight, consistent, and easy to use.

## File Management

### Caja

Caja is the official file manager of the MATE desktop. It allows you to browse directories, preview files, and launch applications associated with them.

**Key Features:**
- Spatial and browser navigation modes
- Tabbed browsing
- Customizable sidebar with bookmarks
- File previews
- Network location support (FTP, SFTP, SMB)
- Extensible through plugins

**Keyboard Shortcuts:**
| Action | Shortcut |
|--------|----------|
| New Window | Ctrl+N |
| New Tab | Ctrl+T |
| Close Tab | Ctrl+W |
| Go Up | Alt+Up |
| Go Home | Alt+Home |
| Show Hidden Files | Ctrl+H |

## Text Editing

### Pluma

Pluma is a small but powerful text editor designed for the MATE desktop. It includes standard text editing features and extends this with other features not usually found in simple text editors.

**Features:**
- Syntax highlighting for many programming languages
- Multiple document editing (tabs)
- Search and replace with regular expressions
- Auto indentation
- Bracket matching
- Line numbers
- Text wrapping
- Current line highlighting
- Plugin system

## Document Viewing

### Atril

Atril is the document viewer for the MATE desktop. It supports multiple document formats including:

- PDF (Portable Document Format)
- PostScript
- DjVu
- TIFF
- DVI
- XPS
- Comic book archives (CBR, CBZ)

**Features:**
- Continuous and dual-page viewing
- Full-screen presentation mode
- Thumbnails sidebar
- Bookmarks
- Text search
- Printing

## Image Viewing

### Eye of MATE

Eye of MATE (eom) is a simple graphics viewer for the MATE desktop. It uses the gdk-pixbuf library and can handle large images.

**Supported Formats:**
- JPEG, PNG, GIF, BMP
- TIFF, ICO
- SVG (with librsvg)
- Many other formats through plugins

**Features:**
- Image slideshow
- Full-screen mode
- Image rotation
- Zoom controls
- EXIF data display

## Archive Management

### Engrampa

Engrampa is an archive manager for the MATE environment. It allows you to create and modify archives, view the contents of an archive, and extract files from an archive.

**Supported Formats:**
- ZIP, TAR, TAR.GZ, TAR.BZ2, TAR.XZ
- 7Z, RAR (extraction only)
- ISO, JAR, CAB
- And many more

## Terminal

### MATE Terminal

MATE Terminal is a terminal emulation application that provides access to a UNIX shell.

**Features:**
- Multiple tabs and windows
- Customizable profiles
- Transparent backgrounds
- Configurable color schemes
- Clickable URLs
- Custom key bindings

## System Tools

### MATE System Monitor

A process viewer and system monitor that shows you what programs are running and how much resources they are using.

**Tabs:**
- **Processes** - View and manage running processes
- **Resources** - CPU, memory, and network usage graphs
- **File Systems** - Disk usage information

### MATE Calculator

A calculator application with basic, advanced, financial, and programming modes.

### MATE Screenshot

A utility for taking screenshots of the entire screen, a window, or a selected area.

## Panel Applets

MATE Panel supports many applets to extend functionality:

- **Clock** - Date and time display with calendar
- **System Monitor** - CPU and memory graphs
- **Weather** - Current weather conditions
- **Notification Area** - System tray for applications
- **Window List** - Shows open windows
- **Workspace Switcher** - Navigate between workspaces
- **Volume Control** - Audio volume adjustment
- **Network Monitor** - Network activity indicator

## Additional Applications

Beyond the core applications, MATE integrates well with:

- **MATE Power Manager** - Battery and power management
- **MATE Sensors Applet** - Hardware temperature monitoring
- **MATE Disk Usage Analyzer** - Visualize disk usage
- **Mozo** - Menu editor for MATE
- **MATE Dictionary** - Look up word definitions
