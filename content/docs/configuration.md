MATE Desktop offers extensive customization options through the Control Center and various configuration tools.

## MATE Control Center

Access the Control Center from the menu: **System → Control Center**

### Appearance

Configure the look and feel of your desktop:

- **Theme** - GTK+ and window border themes
- **Background** - Desktop wallpaper settings
- **Fonts** - System font preferences
- **Interface** - Icon themes and toolbar styles

### Hardware

- **Displays** - Monitor configuration and resolution
- **Keyboard** - Layout and shortcuts
- **Mouse** - Pointer speed and button configuration
- **Power Management** - Battery and power settings

### Personal

- **About Me** - User information
- **Preferred Applications** - Default applications
- **Startup Applications** - Programs that run at login

## Panel Configuration

Right-click on any panel to access configuration options:

- Add or remove panels
- Add applets (clock, system monitor, launchers)
- Change panel position and size
- Configure auto-hide behavior

## Keyboard Shortcuts

Customize keyboard shortcuts in **System → Preferences → Keyboard Shortcuts**:

| Action | Default Shortcut |
|--------|------------------|
| Open Terminal | Ctrl+Alt+T |
| Show Desktop | Ctrl+Alt+D |
| Lock Screen | Ctrl+Alt+L |
| Window Switcher | Alt+Tab |

## Caja File Manager Settings

Configure Caja through **Edit → Preferences**:

- Views (icon, list, compact)
- Behavior (single or double click)
- Display options
- Preview settings
