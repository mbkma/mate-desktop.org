Engrampa is the official archive manager for the MATE desktop. It is a graphical front-end to command-line archiving tools such as tar, zip, and 7z, forked from File Roller. You can create new archives, add and remove files, inspect contents, and extract files — all without touching the terminal.

![Engrampa window](/img/applications/engrampa-window.png "Engrampa archive manager")

## Supported Formats

| Format | Create | Extract |
|--------|--------|---------|
| ZIP | ✓ | ✓ |
| TAR, TAR.GZ, TAR.BZ2, TAR.XZ | ✓ | ✓ |
| 7Z | ✓ | ✓ |
| RAR | — | ✓ |
| ISO | — | ✓ |
| JAR | ✓ | ✓ |
| CAB | — | ✓ |
| AR / DEB | — | ✓ |

## Features

- Create and modify archives with drag-and-drop support
- View archive contents before extracting
- Open individual files inside an archive without extracting the whole thing
- Password-protect ZIP and 7Z archives
- Extract to a chosen folder or directly to the current directory
- Caja integration — right-click files to compress or extract
