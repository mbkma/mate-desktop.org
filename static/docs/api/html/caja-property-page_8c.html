CajaPropertyPage is an object that describes a page in the file properties dialog. Extensions can provide CajaPropertyPage objects by registering a CajaPropertyPageProvider and returning them from caja_property_page_provider_get_pages(), which will be called by the main application when creating file properties dialogs. caja_property_page_new:
