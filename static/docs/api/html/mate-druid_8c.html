mate_druid_construct_with_window: @druid: The MateDruid. @title: A title of the window. @parent: The parent of this window (transient_for). @close_on_cancel: TRUE if the window should be closed when cancel is pressed. @window: Optional return of the #GtkWindow created.

Description: Creates a new toplevel window with the title of @title (which can be NULL) and a parent of @parent (which also can be NULL). The @druid will be placed inside this window. When the druid gets destroyed, so will the window that is created here.
