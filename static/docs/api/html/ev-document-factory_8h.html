ev_document_factory_add_filters: @chooser: a #GtkFileChooser @document: a EvDocument, or NULL

Adds some file filters to @chooser.

Always add a "All documents" format.

If @document is not NULL, adds a #GtkFileFilter for @document's MIME type.

If @document is NULL, adds a #GtkFileFilter for each document type that atril can handle.

ev_document_factory_get_document: @uri: an URI @error: a #GError location to store an error, or NULL

Creates a EvDocument for the document at @uri; or, if no backend handling the document's type is found, or an error occurred on opening the document, returns NULL and fills in @error. If the document is encrypted, it is returned but also @error is set to EV_DOCUMENT_ERROR_ENCRYPTED.

Returns: (transfer full): a new EvDocument, or NULL.
