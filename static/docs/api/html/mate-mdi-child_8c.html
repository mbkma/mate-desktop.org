void mate_mdi_child_set_menu_template ( MateMDIChild *  mdi_child,
MateUIInfo *  menu_tmpl 
)

mate_mdi_child_set_menu_template: @mdi_child: A pointer to a MateMDIChild object. @menu_tmpl: A MateUIInfo array describing the child specific menus.

Description: Sets the template for menus that are added and removed when differrent children get activated. This way, each child can modify the MDI menubar to suit its needs. If no template is set, the create_menus virtual function will be used for creating these menus (it has to return a GList of menu items). If no such function is specified, the menubar will be unchanged by MDI children.
