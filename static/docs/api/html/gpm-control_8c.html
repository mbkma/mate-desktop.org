gpm_control_get_lock_policy: @control: This class instance @policy: The policy string.

This function finds out if we should lock the screen when we do an action. It is required as we can either use the mate-screensaver policy or the custom policy. See the yelp file for more information.

Return value: TRUE if we should lock.
