Enumeration Type Documentation

◆ MateEdgePosition

enum MateEdgePosition

MateEdgePosition: @MATE_EDGE_START: The current page is at the beginning of the druid. @MATE_EDGE_FINISH: The current page is at the end of the druid. @MATE_EDGE_OTHER: The current page is neither the first nor the last page (usually not required). @MATE_EDGE_LAST: Used internally to indicate the last value of the enumeration. This should not be passed in to any function expecting a MateEdgePosition value.

Used to pass around information about the position of a MateDruidPage within the overall MateDruid. This enables the correct "surrounding" content for the page to be drawn.
