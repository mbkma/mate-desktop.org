31/****
32 Sugar functions to pop up dialogs in a hurry. These are probably
33 too sugary, but they're used in mate-app-util anyway so they may as
34 well be here for others to use when there's no MateApp.
35 The mate-app-util functions are preferred if there's a MateApp
36 to use them with, because they allow configurable statusbar messages
37 instead of a dialog.
38 ****/
