◆ cact_menubar_new()

CactMenubar * cact_menubar_new ( BaseWindow *  window)

cact_menubar_new: @window: the window which embeds the menubar, usually the CactMainWindow.

The created menubar attachs itself to the @window; it also connect a weak reference to this same @window, thus automatically g_object_unref() -ing itself at @window finalization time.

The menubar also takes advantage of BaseWindow messages to initialize its Gtk widgets.

Returns: a new CactMenubar object.
