Detailed Description

How to draw frames at different times: when it's maximised or not, shaded or not, when it's focussed or not, and (for non-maximised windows), when it can be horizontally or vertically resized, both, or neither. Not all window types actually get a frame.

A theme contains one of these objects for each type of window (each MetaFrameType), that is, normal, dialogue (modal and non-modal), etc.

This corresponds closely to the <frame_style_set> tag in a theme file.
