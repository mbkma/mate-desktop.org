typedef GSList *(* pget_filters) (void *)

SECTION: cact_match_list @short_description: Implementation of a list match/does not match. cact/cact-match-list.h

In an ideal world, this would be a base interface for CactISchemesTab, etc. interfaces. In GObject world however, one cannot derived an interface, nor an interface can implement another interface. The GObject solution would be for CactISchemesTab to requires CactIMatchList, and for CactMainWindow to implement this same CactIMatchList interface. This is not very practical as CactMainWindow is already some big bunch of code...

So we stay with just a piece of helper functions...
