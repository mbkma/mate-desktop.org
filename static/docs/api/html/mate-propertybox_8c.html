GtkWidget * mate_property_box_new ( void  )

mate_property_box_new: [constructor]

Creates a new MatePropertyBox widget. The PropertyBox widget is useful for making consistent configuration dialog boxes.

When a setting has been made to a property in the PropertyBox your program needs to invoke the mate_property_box_changed to signal a change (this will enable the Ok/Apply buttons).

Returns a newly created MatePropertyBox widget.
