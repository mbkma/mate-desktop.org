42/* The MateColorPicker widget is a simple color picker in a button. The button displays a sample
43 * of the currently selected color. When the user clicks on the button, a color selection dialog
44 * pops up. The color picker emits the "color_set" signal when the color is set
45 *
46 * By default, the color picker does dithering when drawing the color sample box. This can be
47 * disabled for cases where it is useful to see the allocated color without dithering.
48 */
49
