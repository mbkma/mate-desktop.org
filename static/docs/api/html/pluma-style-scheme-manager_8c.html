_pluma_style_scheme_manager_install_scheme: @manager: a #GtkSourceStyleSchemeManager @fname: the file name of the style scheme to be installed

Install a new user scheme. This function copies @fname in #PLUMA_STYLES_DIR and ask the style manager to recompute the list of available style schemes. It then checks if a style scheme with the right file name exists.

If the call was succesful, it returns the id of the installed scheme otherwise NULL.

Return value: the id of the installed scheme, NULL otherwise.
