na_tray_child_set_composited; @child: a NaTrayChild @composited: TRUE if the child's window should be redirected

Sets whether the #GdkWindow of the child should be set redirected using gdk_window_set_composited(). By default this is based off of na_tray_child_has_alpha(), but it may be useful to override it in certain circumstances; for example, if the NaTrayChild is added to a parent window and that parent window is composited against the background.
