SECTION: cact_iaction_tab @short_description: CactIActionTab interface definition. cact/cact-iaction-tab.h

This interface implements the "Caja Menu Item" tab of the notebook.

Entry fields are enabled, as soon as an edited item has been set as a property of the main window, Fields are those of NAObjectItem (i.e. not NAObjectProfile, not NAIContext, but may be NAObjectAction specific).
